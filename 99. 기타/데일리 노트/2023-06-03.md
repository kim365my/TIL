# 오늘 배운 것
- 브라우저 객체 모델
	- window
		- document : [[HTML]]문서를 표현 (DOM 객체의 최상위 객체)
		- history : 사용자가 방문한 URL을 저장, 관리
		- location : 현재 URL에 대한 정보를 유지
		- navigator : 현재 사용 중인 브라우저에 대한 정보를 관리
		- screen : 스크린 장치에 대한 정보관리
