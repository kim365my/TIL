💡 **왜 사용하는가?**
- 멀티브라우져 지원, 이벤트+애니메이션+복잡한 CSS 변환+AJAX의 기능 모두를 다양하게 사용할 때, 그리고 위의 기능들을 구현할만한 시간적 여유가 없고 jquery에 대해서 이미 알고 있을 때. 그리고 무엇보다도 페이지가 퍼포먼스를 크게 요구하지 않고 사용자와의 인터렉션이 많지 않을 때 사용
- 장점
    - 멀티브라우저 지원
    - 간단한 이벤트 설정, DOM 탐색
- 단점
    - jquery API를 별도로 공부
    - 퍼포먼스(32k 용량)의 문제 
    - 가상 DOM의 등장
