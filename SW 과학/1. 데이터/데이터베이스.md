-   **데이터베이스** : 자료저장방식
    1.  자료를 저장하는 원시적인 방법 : 한 파일에 모든 자료를 저장 → 보안 위험!!
    2.  관계형 데이터 베이스(Relational Database, RDB) : 표로 저장하는 체계(ex. 시간표)
-   **데이터베이스의 종류**
    1.  Relational database ([[SQL]])
        -   mysql
        -   postgresql
        -   sqlite
    2.  Non-Relational database (Non-SQL)
        1.  mongo DB
        2.  Dynamo DB
        3.  couch DB
